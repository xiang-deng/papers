Contextual information in search sessions is important for capturing users search intents. Various approaches have been proposed to model user behavior sequences to improve document ranking in a session. Typically, training samples of (search context, document) pairs are sampled randomly in each training epoch. In reality, the difficulty to understand user s search intent and to judge document s relevance varies greatly from one search context to another.
