A grande quantidade de dados hoje gerada pela Internet e por organizacoes requer novas abordagens de gerenciamento desses dados. Data lakes sao repositorios de dados volumosos e tem sido uma alternativa para gerenciar dados e metadados heterogeneos. Este trabalho apresenta um estudo sobre arquiteturas e abordagens para gerenciamento de metadados no contexto de sistemas de gerencia de data lake. Cites: Goods: Organizing google s datasets
