Aspect-based sentiment analysis (ABSA) aims at identifying the opinion aspects (aspect extraction) and sentiment polarities toward corresponding aspects (sentiment classification) from a sentence. Recently, some span-based methods, which first extract aspects by detecting aspect boundaries and then predict the span-level sentiments, have achieved promising results.
