The task of question answering is to find the most appropriate answer for an input question in natural language from a given custom knowledge base of information. While the performance of question answering systems has been significantly improved, they still struggle to answer questions that require commonsense reasoning. To capture common sense beyond associations, a challenging dataset CommonsenseQA for commonsense question answering is proposed.
