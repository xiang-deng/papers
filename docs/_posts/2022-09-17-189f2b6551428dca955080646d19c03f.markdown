Constructing a personalized end-to-end task-oriented dialogue system is one of the most important and challenging tasks in natural language processing technology. Slot-filling has achieved success in a rule-based task-oriented dialogue system. However, building a rule-based task-oriented dialogue system for real conversations is time-consuming. We present a novel personalized end-to-end framework based on split memory for Memory Networks by using topic model in this paper.
