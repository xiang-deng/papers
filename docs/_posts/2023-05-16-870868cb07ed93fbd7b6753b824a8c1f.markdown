As cidades inteligentes so um contexto que pode obter grande vantagem no formato e no crescimento de dados na web semntica, visto que o volume e a conexo aumentam a qualidade das anlises de dados. No entanto, o crescimento quantitativo de dados deve acontecer com garantia de qualidade. Este trabalho tem como objetivo a verificao de qualidade de dados na fuso de dados conectados, por meio das dimenses de qualidade acurcia, consistncia e conciso.
