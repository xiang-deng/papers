In real world, Database stores the information of object. The management of database system provide to load, access, and manipulate the information. Data stored in database is either certain and uncertain. The certain data are those data whose value is known and fixed. If the certain attribute value is not known, then it will have treated as Null value. The uncertain data is data whose value is unknown and not fixed.
