Digitized historical newspaper collections create unique opportunities and challenges in the realm of big data analysis systems. For domains like the social sciences and journalism, the availability of these large data collections can provide researchers (and the public) with the resources to ask broader questions of historical relevance. This type of information, if available, was previously only accessible by visiting a library or an archive and performing extensive manual research.
