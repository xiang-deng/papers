In this work we present a new AI-based UI for question answering systems when the answer is a list. We use AI to analyze list answers so that we can provide additional information in the UI to enhance user experience. We explore the list type, the introductory text, and the size of the list. We conduct a user-experience study with an indirect and direct evaluations to show the impact of this novel list presentation.
