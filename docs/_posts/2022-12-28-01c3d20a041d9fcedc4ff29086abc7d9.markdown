Sentiment-controlled text generation aims to generate texts according to the given sentiment. However, most of the existing studies focus only on the document-or sentence-level sentiment control, leaving a gap for finer-grained control over the content of generated results. Fine-grained control allows a generated review to express different opinions toward multiple aspects.
