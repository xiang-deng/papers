Aspect or query-based summarization has recently caught more attention, as it can generate differentiated summaries based on users interests. However, the current dataset for aspect or query-based summarization either focuses on specific domains, contains relatively small-scale instances, or includes only a few aspect types. Such limitations hinder further explorations in this direction.
