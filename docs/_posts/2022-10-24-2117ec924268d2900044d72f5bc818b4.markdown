Natural Language to SQL (NL2SQL) is a technology that converts natural language into executable SQL statements. For single table tasks, the dominant approach is a slot-filling-based model which actually performs multi-objective prediction. However, in existing multi-objective models, it is difficult to optimize multiple objectives simultaneously. Low correlation or conflict between various objectives can result in the model not being able to learn the parameters effectively.
