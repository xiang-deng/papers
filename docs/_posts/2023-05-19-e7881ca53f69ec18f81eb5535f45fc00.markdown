Every smart home user interaction has an explicit or implicit goal. Existing home assistants easily achieve explicit goals, eg, turn on the light . In more natural communication, however, humans tend to describe implicit goals. We can, for example, ask someone to make it cozy rather than describe the specific steps involved. Current systems struggle with this ambiguity since it requires them to relate vague intent to specific devices.
