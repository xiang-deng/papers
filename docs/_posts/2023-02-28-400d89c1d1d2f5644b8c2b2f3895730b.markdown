In open domain question answering systems, Dense Passage Retriever (DPR) has become a popular approach to retrieving relevant passages for finding answers. However, the biggest shortcoming of such a system is it takes too much memory to store the index and model. Though Binary Passage Retriever (BPR) significantly reduces the size of the index, it still suffers from the heavy model size because of large pre-trained models.
