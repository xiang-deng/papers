Abductive reasoning aims to make the most likely inference for a given set of incomplete observations. In this work, given a situation or a scenario, we aim to answer the question what is the set of actions that were executed by the human in order to come to this current state? , which we coin as abductive action inference. We provide a solution based on the human-object relations and their states in the given scene.
