Abstract Aspect-category-opinion-sentiment (ACOS) quadruple extraction is a fine-grained sentiment analysis task to extract full sentiment information, which aims to extract all the ACOS quads in a given sentence. ACOS contains four types of quads: explicit aspect and explicit opinion, implicit aspect and explicit opinion, explicit aspect and implicit opinion, and implicit aspect and implicit opinion.
