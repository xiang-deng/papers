Dialogue skills are crucial in everyday life, and machine learning tools that model and analyze data can be essential in helping humans with dialogue-based tasks. However, most dialogue models rely solely on text data, given the lack of spoken dialogue datasets. In this thesis, we build the DARE interface, which is designed to collect spoken dialogues.
