This paper presents an overview of the NLPCC 2022 shared task 5 track 2, ie, Named Entity Recognition (NER), which aims at extracting entities of interest from domain-specific texts (material science). The task provides 5600 labeled sentences (with the BIO tagging format) collected from ACS material science publications.
