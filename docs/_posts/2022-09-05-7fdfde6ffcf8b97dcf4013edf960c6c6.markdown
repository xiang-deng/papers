In this paper, we present Tetris, a new task of Goal-Oriented Script Completion. Unlike previous work, it considers a more realistic and more general setting, where the input includes not only the goal but also additional user context, including preferences and history. To address the problem using a knowledge-based approach, we introduce Task Concept Graph, an automatically constructed knowledge base from instructional websites.
