Multi-review summarization, the process of automatically generating a summary for a set of reviews, is of value to customers when they are making informed decisions. Incorporating sentiments can improve the performance of abstractive review summarization, as shown in some prior studies. However, these studies assume that each review contains a single sentiment, which does not reflect reality in many real conditions.
