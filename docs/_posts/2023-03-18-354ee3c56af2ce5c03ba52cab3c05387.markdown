Abstract The Threshold-based Nave Bayes (Tb-NB) classifier is introduced as a (simple) improved version of the original Nave Bayes classifier. Tb-NB extracts the sentiment from a Natural Language text corpus and allows the user not only to predict how much a sentence is positive (negative) but also to quantify a sentiment with a numeric value.
