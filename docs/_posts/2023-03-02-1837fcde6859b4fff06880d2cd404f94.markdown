Keyphrase extraction is a fundamental task in Natural Language Processing (NLP) that aims at extracting a set of important phrases from a source document. Generally, when people understand documents and extract relevant information, they consider multiple perspectives, which helps to reduce misunderstandings and errors in extractions. However, most existing keyphrase extraction approaches focus on only one or two perspectives, resulting in inaccurate extractions.
