摘要为解决从未标记文本中自动化的生成问答对(Q&A), 提出了一种基于Seq2Seq 框架和自动化构建数据的问答对生成方法(Seq2Seq-TDP-QAG). 首先, 已知段落, 问题, 答案的阅读理解数据集中, 通过构造函数从中获得线索, 问题类型信息构造训练数据集, 并用数据训练基于Seq2Seq 框架的问题生成模型; 其次, 采用三步取样法从未标记文本中获取答案, 线索, 问题类型信息, 构造问题生成模型的输入数据集, 并输入模型中初步生成问题信息; 最后, 通过数据过滤器来控制问题的质量, 生成高质量的问题信息 …
