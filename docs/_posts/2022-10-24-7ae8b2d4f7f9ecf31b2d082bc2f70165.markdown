It is common for a web page to include links which help visitors discover related pages on other sites. When a link ceases to work (eg, because the page that it is pointing to either no longer exists or has been moved), users could rely on an archived copy of the linked page. However, due to the incompleteness of web archives, a sizeable fraction of dead links have no archived copies. We study this problem in the context of Wikipedia.
