Flashcards are the main tool used in the Spaced Repetition memorization method, yet there are not always available for many topics due to the high effort required to create them. The combination of Transformer-based models with a Recommender System (RS) can enable a dynamic model to auto generate flashcards recommendation for learners and serious game players in order to improve their skills in learning programming.
