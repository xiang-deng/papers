Narratives determine our worldviews, but need evidence to be believable. For scientific narratives, hard evidence usually is provided by specially curated research data and experiments within each publication. The aim is to strengthen the narratives overall plausibility: the less plausible a narrative is, the more it is bound to be challenged.
